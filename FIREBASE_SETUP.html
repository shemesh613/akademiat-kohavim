🔥 הגדרת Firebase — 5 דקות

צריך פרויקט Firebase ריק כדי שילדים יוכלו להתחבר מהבית. זה חינם.

1צור פרויקט
היכנס ל-console.firebase.google.com
לחץ "Add project" → תן שם (למשל akademiat-kohavim) → Continue
בטל Google Analytics (לא צריך) → Create project
2הפעל Realtime Database
בתפריט השמאלי: Build → Realtime Database → Create Database
בחר europe-west1 (קרוב לישראל) → Next
בחר Start in test mode → Enable
3הוסף "אפליקציית web"
חזור לעמוד הראשי של הפרויקט (סמל הבית בצד שמאל למעלה)
לחץ על סמל </> (Web) שמופיע בעמוד
תן כינוי (למשל kohavim-web) → Register app
תקבל בלוק קוד שנראה ככה: 
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "...firebaseapp.com",
  databaseURL: "https://...firebaseio.com",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
4שלח לי את ה-config
העתק את כל הבלוק (מ-const firebaseConfig = { עד };) והדבק לי כאן בצ'אט.
אני אטמיע את זה באפליקציה ואסיים את הסנכרון.
🔒 אבטחה: בהמשך נשנה את ה-Database Rules כדי שתלמיד יוכל לקנות רק לעצמו. נטפל בזה אחרי שהקוד מותקן.
✅ אחרי שתשלח לי את ה-config, אני:
  • אטמיע את Firebase באפליקציה
  • אוסיף לכל תלמיד טוקן אישי + כפתור 📲 ליצירת קישור
  • אבנה מסך תלמיד נפרד שמראה רק את הפרופיל שלו
  • אגדיר חוקי אבטחה כך שתלמיד לא יוכל להוסיף ניקוד לעצמו (רק לקנות)
