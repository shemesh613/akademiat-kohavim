🔓 תיקון חוקי Firebase (30 שניות)

ה-Database שלך נעול — כל ניסיון קריאה מקבל "Permission denied" 401. הסיבה: בחרת בטעות "locked mode" במקום "test mode". מתקנים בקליק.
1 חזור ל-Firebase Console → פרויקט nikud
2 בתפריט שמאל: Realtime Database
3 למעלה תראה 3 לשוניות: Data | Rules | Backups
לחץ על Rules
4 תראה משהו כזה: 
{
  "rules": {
    ".read": false,
    ".write": false
  }
}
החלף את כל התוכן ב: 
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
5 לחץ Publish בכחול למעלה
תקבל אישור "Rules published"
🔒 לא מסוכן: ה-URL שלך גלוי רק ב-HTML של האתר שלך. אף אחד לא יודע לחפש אותו אקראית.
