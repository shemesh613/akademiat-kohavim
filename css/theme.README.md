# theme.css — שפת "יום שמש בכיתה"

## שורת החיבור ל-index.html
להוסיף **מיד אחרי** שורת ה-link של shop-plus.css (כיום שורה 761):

```html
<link rel="stylesheet" href="css/theme.css">
```

סדר הטעינה חובה: `<style>` הפנימי ← `css/shop-plus.css` ← `css/theme.css` (אחרון).
הקובץ דורס בקסקדה רגילה — אין תלות ב-JS ואין שינויי מרקאפ.

## מה עוצב (theme.css)
- **טוקנים**: 5 צבעי היסוד (זהב/אלמוג/שמיים/עלה/ענב) + גוון-צל כהה לכל אחד, נייטרלים, שפת עץ-קרם של האי, סקאלת 8 גדלי טקסט, רדיוסים (10/16/24/32/999), 3 רמות צל רך.
- **רקע עם עומק**: גרדיאנט שמיים + גבעות ירוקות עדינות בתחתית (body::after, CSS בלבד) + חלקיקים מרחפים בצבעי היסוד.
- **כותרת עליונה**: קורת עץ בהירה עם עורקי-עץ, לוגו "שלט זהב" (בלי קשת ניאון), טאבים כשלטי עץ — הפעיל נצבע זהב.
- **לוח הכיתה**: עמודות כ"לוחות שעם" קרם עם נקודות-שעם ומסגרת עץ 3px; כותרות עמודה כשלטי עץ תלויים עם צל תחתון קשיח.
- **כרטיס תלמיד** (הרכיב שמופיע 24 פעמים): פתק לבן במסגרת עץ, שם 23px, נקודות כ"מטבע זהב" (שבב מלא, לא טקסט-גרדיאנט שקוף), אווטאר עם טבעת-מדבקה (box-shadow בלבד — לא נוגעים בממדי ה-canvas), hover עם lift.
- **חגיגה ואכזבה**: `.celebrating` — קפיצת כרטיס, שטיפת זהב, זוהר וקונפטי 🎉; `.disappointed` — מסגרת אלמוג וטיפת דמעה 💧. רצף `.on-fire` בגבולות מלאים; זוהר אמיתי רק ב-tier4.
- **כפתורים**: "כפתורי צעצוע" — מילוי מלא, מסגרת 3px, צל תחתון קשיח שנראה לחיץ, active שמכווץ. חמשת כפתורי הנקודות ממופים לחמשת צבעי היסוד.
- **מודלים**: גוף קרם עם מסגרת עץ ורדיוס 32, כותרת 36px, טאב פעיל זהב, תיבת ההפתעה כשלט ענב.
- **מקרן**: כל טקסט קריטי ≥22px משקל כבד; דריסת inline של `font-size:1.1em` בכפתורי-הדגל (!important נקודתי הכרחי מול inline); אפס גבולות אפור-דק-שקוף.
- **פופאפים** (נקודות/קומבו): קו-מתאר לבן במקום זוהר ניאון — קריא על הרקע הבהיר.
- **נגישות**: `prefers-reduced-motion` מכבה את כל התנועה; RTL מלא (inset-inline, צללים אנכיים בלבד).

## מה עוצב (shop-plus.css — שוכתב לשפה הבהירה)
- כרטיסי פריט כ"מדבקות" לבנות עם מסגרת עץ וצל לחיץ; ברק חולף ב-hover.
- **נדירות במסגרות צבע מלאות**: rare=שמיים, epic=ענב, mythic=זהב — **זוהר נושם ל-mythic בלבד** (כולל הילה מאחורי התצוגה המקדימה).
- **מצב נעול מכובד**: מסגרת מקווקוּה + שבב מנעול בפינה + תג מחיר כ"יעד חיסכון" בזהב — לא אפרורי-עצוב.
- **אנימציית קנייה**: הכרטיס "נחתם" כמדבקה שהודבקה (pop) וכוכב ⭐ קופץ בפינת הפריט הלבוש.
- **סרגלי אוסף** כ"פס מדבקות" קרם: טקסט 22px, פס התקדמות עבה במילוי זהב עם נצנוץ; אחוזים מוכהים לקריאות מקרן.
- שם פריט 22px בדיו כהה, שבבי מחיר/סטטוס מלאים (זהב/עלה) עם טקסט מנוגד.
